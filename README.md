# 상품명 파서 (Product Name Parser)

골프 용품 주문목록 엑셀 파일의 상품명을 **브랜드 / 메인모델 / 서브모델 / 카테고리 / 기타태그**로 자동 분류하는 Streamlit 앱입니다.

---

## 설치 및 실행

### 1. 사전 요구사항
- Python 3.9 이상

### 2. 패키지 설치

```bash
pip install -r requirements.txt
```

### 3. 실행

```bash
streamlit run app.py
```

또는 Windows에서는 `run.bat` 더블클릭으로 실행 가능합니다.

실행 후 브라우저에서 `http://localhost:8501` 접속

---

## 앱 구성

| 페이지 | 설명 |
|--------|------|
| **메인** | 엑셀 업로드 → 파싱 → 결과 확인 → 다운로드 |
| **사전관리** | 브랜드 / 모델 / 카테고리 추가·수정·삭제 |
| **미분류검토** | 분류 안 된 항목 확인 및 사전 등록 |

---

## 사용 방법

### 엑셀 업로드 및 파싱
1. 메인 화면에서 주문목록 엑셀 파일(`.xlsx`, `.xls`) 업로드
2. **상품명** 컬럼이 자동 감지됩니다 (감지 안 되면 직접 선택)
3. **파싱 시작** 버튼 클릭
4. 결과 테이블 확인 후 **엑셀 다운로드**

### 사전 관리
- **브랜드 탭**: 브랜드 추가 시 `name`(표시명)과 `aliases`(별칭, 쉼표 구분) 입력
  - 예) name: `젝시오`, aliases: `XXIO, xxio`
- **모델 탭**: 특정 브랜드 선택 후 메인모델 / 서브모델 추가
- **카테고리 탭**: 카테고리 추가 시 `priority` 값 지정 가능

### 미분류 검토
- 브랜드 또는 카테고리가 매칭되지 않은 항목 목록 확인
- 직접 분류 입력 → **사전에 추가** 버튼으로 등록
- 등록 후 다음 파싱부터 자동 분류됨

## 분류 결과 컬럼

| 컬럼 | 설명 |
|------|------|
| 원본상품명 | 원래 상품명 |
| 브랜드 | 매칭된 브랜드명 |
| 메인모델 | 브랜드 소속 메인모델 |
| 서브모델 | 메인모델 소속 서브모델 |
| 카테고리 | 드라이버 / 퍼터 / 골프공 / 가방 / 커버 등 |
| 태그 | 남성, 여성, 시타채, 좌타, 리퍼브, 전시상품, 임직원 |
| 분류상태 | 완료 / 부분 / 미분류 |
